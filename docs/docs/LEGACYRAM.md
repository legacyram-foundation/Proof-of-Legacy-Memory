# LEGACYRAM Network

LEGACYRAM is the first blockchain designed to operate under the
Proof of Legacy Memory (PoLM) consensus.

The network aims to revive legacy hardware worldwide,
transforming abandoned computers into active participants
of a decentralized computational infrastructure.

Mission:
Revive hardware.
Democratize computation.
Preserve technological history.
